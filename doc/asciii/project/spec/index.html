asciii::project

[−][src]Module asciii::project::spec

Implements the ascii invoicer project file specification.

This does all of the heavy lifting. The implementation is separated into sub-modules which take care of separate objectives. Most of the functions in these modules take the yaml data directly as reference. Each module contains a validate() function which ought to be kept up to date.
